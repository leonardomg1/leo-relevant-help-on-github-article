## Por que este artigo existe

Muitas pessoas querem ajudar projetos open source (código aberto), mas não sabem por onde começar.

Este artigo mostra um caminho simples: começar com **pequenas e úteis contribuições**, como traduzir um README, melhorar a documentação, corrigir instruções pouco claras, revisar links ou ajudar as pessoas a encontrar problemas simples (issues para iniciantes).

Ele trata a contribuição em código aberto como **filantropia digital**: uma forma de doar tempo, atenção, clareza e acesso a pessoas que você talvez nunca conheça.

## Comece a ajudar agora

Use estas buscas do GitHub para encontrar projetos que possam precisar de ajuda.

### Tradução e localização (i18n)

- [Projetos com `locales en.json` e sem `pt-BR`](https://github.com/search?q=locales+en.json+NOT+pt-BR&type=code)
- [Projetos com `i18n en.json` e sem `pt-BR`](https://github.com/search?q=i18n+en.json+NOT+pt-BR&type=code)
- [Issues de tradução pedindo ajuda](https://github.com/search?q=translation+help+wanted&type=issues)
- [Issues de localização mencionando Português](https://github.com/search?q=localization+Portuguese+help+wanted&type=issues)
- [Good first issues sobre i18n](https://github.com/search?q=i18n+good+first+issue&type=issues)

### Contribuições amigáveis para iniciantes

- [Good first issue + documentação](https://github.com/search?q=good+first+issue+documentation&type=issues)
- [Help wanted + documentação](https://github.com/search?q=help+wanted+documentation&type=issues)
- [Good first issue + tradução](https://github.com/search?q=good+first+issue+translation&type=issues)
- [Help wanted + localização](https://github.com/search?q=help+wanted+localization&type=issues)

Mais buscas selecionadas estão disponíveis em [references/github-search-links.md](./references/github-search-links.md).

## O que você pode melhorar aqui

Este artigo aceita pequenas e cuidadosas contribuições:

- melhorar a clareza;
- adicionar melhores links de busca;
- corrigir links quebrados;
- traduzir para outro idioma;
- revisar uma tradução existente;
- sugerir exemplos reais de ajuda amigável para iniciantes em open source.

Por favor, leia o arquivo [CONTRIBUTING.md](./CONTRIBUTING.md) antes de abrir um pull request.
