# Contribuindo com o Artigo

Obrigado por ajudar a aprimorar este artigo de filantropia digital!

## Como Participar

### 1. Corrigindo Erros ou Sugerindo Melhorias
Se você encontrar erros de digitação, links quebrados ou tiver sugestões para melhorar o texto:
- Abra uma issue usando nossos templates de Issue na pasta `.github/ISSUE_TEMPLATE/`.
- Ou envie um Pull Request diretamente alterando um dos arquivos `helping-open-source-through-digital-philanthropy.*.md`.

### 2. Enviando uma Nova Tradução
Quer traduzir o artigo para outro idioma (ex: Italiano, Francês, Alemão)?
1. Crie uma cópia do arquivo `helping-open-source-through-digital-philanthropy.md` (em inglês) ou `helping-open-source-through-digital-philanthropy.pt-BR.md` (em português).
2. Nomeie o novo arquivo como `helping-open-source-through-digital-philanthropy.<CÓDIGO_DO_IDIOMA>.md` (ex: `helping-open-source-through-digital-philanthropy.it.md` para italiano).
3. Adicione o novo idioma e tradução nos metadados em [metadata/helping-open-source-through-digital-philanthropy.yml](metadata/helping-open-source-through-digital-philanthropy.yml).
4. Submeta o Pull Request.

Agradecemos profundamente sua colaboração!
